UX Design Methods

User Strategy and Design Methods from uxdesign.cc. I comment on the ones that I’ve been experienced through before:

Design Strategy Methods

  1. Blueprint
  2. Consumer Journey Map

    A consumer journey map tells the story of the customer’s experience: from initial contact, through the process of engagement and into a long-term relationship.


    Before going down to functions and details, a consumer journey map helps me plan a complete user process first. This process starts from where the user gets interested in the product to how company builds a long-term relationship with the user after-sales.
  3. User Stories
    After listing out the consumer journey map, describing the user stories help you deeper understand user behavior and user need in a specific scenario.
  4. Personas
    Personas is a crucial part for UX design. It helps you deeper understand your target user. Everytime you don’t know how to decide, look at your personas.
    5 Ecosystem Map
  5. Competitive Audit
  6. Value Proposition
  7. Stakeholders Interviews
  8. Key Performance Indicators

Concepting Methods

  1. Brainstorming
  2. Moodboards
  3. Storyboards
  4. User Flow
    Walk through your design as your user. A fluid user flow is very important to make the whole experience pleasant.
  5. Task Analysis
  6. Taxonomies

Product Planning Methods

  1. Content Audit
  2. Heuristic Analysis
  3. Sitemap
    I find sitemap super useful for designing the layout for each page. When you know what functions you need on each page, you will never get lost.
  4. Features Roadmap
  5. Use Cases and Scenarios
  6. Metrics Analysis

User Researh & Validation Methods

  1. Focus Group
    Focus group is a good way to let user discuss, or even argue.
  2. Quantitative Survey
  3. Usability Test
  4. Card Sorting
    Card sorting is very helpful for a team to sort out ideas.
  5. A/B Test
  6. Eyetracking
  7. Accessibility Analysis
